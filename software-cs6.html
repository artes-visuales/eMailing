Si ha adquirido un producto CS6, y ya no tiene los discos de instalación, puede descargar instaladores de sustitución CS6 de esta página.

Lea esto antes de descargar el software CS6

  • Necesitará el número de serie de CS6 para realizar la instalación. El número de serie se lo enviaremos por mail una vez Adobe lo asigne.
    Hasta ese momento debe realizar los siguientes pasos:
    Acceda al enlace inferior CS6 Descarga de productos.
  • En el caso de OSX el proceso de instalación es el siguiente:
    • Cuando aparezca la ventana Guardar, acceda a la carpeta en la que desea guardar el archivo descargado.
    • Empezará a descargarse un archivo DMG. Espere hasta que el archivo DMG se haya descargado para empezar el proceso de instalación.
    • Una vez que se haya descargado el archivo DMG, aparecerá un cuadro de diálogo en el que se le preguntará si desea abrir el archivo DMG. Haga clic en Aceptar. Si no aparece este cuadro de diálogo, acceda a la ubicación en la que guardó el archivo DMG descargado. A continuación, haga doble clic en el archivo para abrirlo.
    • Es posible que aparezca un cuadro de diálogo con el mensaje "[Nombre del producto de Adobe] es un archivo descargado de Internet. ¿Está seguro de que desea abrirlo?" Seleccione Sí.
    • Una vez finalizada la verificación del archivo DMG, se abrirá una nueva ventana con el programa de instalación.
    • Antes de continuar con la instalación, cierre todas las aplicaciones activas en el sistema (incluidas otras aplicaciones de Adobe o de Microsoft Office y las ventanas del navegador).
    • Haga doble clic en el icono del producto para abrir la carpeta del producto y, a continuación, haga doble clic en el icono de instalación.
    • Siga las instrucciones que aparecen en pantalla para completar la instalación.
    • Cuando le diga si dispone de número de serie o quiere instalar un trial elija trial.
    El el caso de PC es similar pero cambiando la extensión del fichero.
  • Asegúrese de que el sistema cumple los requisitos para productos de Creative Suite 6.
  • ¿Tiene problemas? Obtener ayuda de nuestros expertos de la comunidad.
